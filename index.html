What is this?

This is a practice tool that simulates frame perfect strategies from certain levels in the Happy Wheels demo. The graphics are built from hundreds of screenshots of the levels being played in the flash demo of the real game. This tool allows for better practice through its added features that aren't available in the real game. These features include save states for retrying specific scenarios instantly, custom framerate for slowing down or speeding up the simulation, pausing on failed inputs to see exactly what frame the error occurred on, and input feedback to see what caused the error.
